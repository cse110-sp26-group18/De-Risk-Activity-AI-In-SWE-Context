## USER STORIES

### 1. Former Gambler
**As a former gambler,**   
I want an app that satisfies my gambling urges and provides a dopamine rush so that I can feel like I’m truly at a casino without real risk. I also want to be able to play more than just a few spins per day, since other slot machine apps tend to limit gameplay too much.

---

### 2. Rational Player
**As a rational person,**   
I want to play multiple rounds every day for fun and to test my luck so that I can pass time in an enjoyable way without expecting real winnings. I also need a steady supply of daily tokens and a comfortable interface that doesn’t feel overwhelming like a real casino.

---

### 3. Customization-Seeking Player
**As a casual player after work,**   
I want to unlock new themes and rewards so that my experience feels fresh and not repetitive. I also want to be able to play anywhere conveniently, while still getting a similar vibe to a real casino.

---

### 4. Competitive Player
**As a competitive player,**   
I want a ranking system so that I can compare my performance with friends and stay motivated. I also want a starter package from referrals, along with social features like sharing wins and collaborating with friends to enhance the experience.

---

### 5. Bored Student User
**As a bored student in a lecture,**   
I want a simple, fast-paced, high-excitement game so that I can quickly relieve boredom without committing to long sessions or dealing with a learning curve.
